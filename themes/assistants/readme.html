Building an Assistant

Discover how to create and manage AI assistants on this platform easily.
Learn about advanced settings and customization options.

Overview

Building an AI assistant on this platform is intuitive. You can access available assistants or create a new one via the assistant section.

Creating a New Assistant

If you wish to create a new assistant, click the "Create new assistant" button. You'll be asked to provide a name, description, and an optional image.

Advanced Settings

In the advanced settings, you can configure options like temperature, context length, and whether profile context or workspace instructions are included in chats.

Summary

Building and managing AI assistants is smooth and flexible, providing a streamlined experience for managing tasks and interactions.
