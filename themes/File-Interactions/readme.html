How to Upload and Use Files

Last updated 10/5/2024

Overview

The file feature allows you to upload and manage files on the platform. You can attach files for various uses, such as sharing documents or providing additional context for AI interactions.

Step-by-Step Guide

1. Uploading a File

To upload a file:

    Click the Create File button to begin.

    Click the Choose File button and select the file from your device.

2. Naming Your File

After selecting your file:

    In the Name field, enter a clear name for your file (e.g., "Project Proposal").

3. Adding a Description

Optionally, you can add a description for your file:

    In the Description field, provide a brief summary of the fileâ€™s contents (e.g., "This file contains the project proposal for client XYZ").

4. Saving the File

Once you have named and described your file:

    Click Save to upload and store the file in the platform for future use.

Using Uploaded Files

Once uploaded, your files can be used in various contexts, such as attaching them to conversations or referencing them in different tasks.
