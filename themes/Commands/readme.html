Commands

Overview

The commands feature allows you to quickly interact with the platform using shortcuts like "/", "@", "#", and "!". These commands help you navigate through prompts, assistants, collections, and tools.

Command Usage

Use "/" for prompts, "@" for assistants, "#" for collections, and "!" for tools.

Summary

Commands make it easier to interact with the platform, allowing quick access to key features through simple shortcuts.
