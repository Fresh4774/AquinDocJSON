Shortcuts

Last updated 10/6/2024

Overview

Shortcuts allow you to perform common tasks quickly and efficiently. The platform offers several predefined keyboard shortcuts to help you navigate, manage files, open settings, and toggle various features with ease.

Available Shortcuts

    Show Help: CTRL + /

    Show Workspaces: CTRL + ;

    New Chat: CTRL + O

    Focus Chat: CTRL + L

    Toggle Files: CTRL + F

    Toggle Retrieval: CTRL + E

    Open Settings: CTRL + S

    Toggle Sidebar: CTRL + B

Customization

Depending on your workflow, you can customize shortcuts to better match your preferences and increase efficiency.

Summary

Shortcuts offer a streamlined way to navigate the platform, manage tasks, and interact with key features quickly and efficiently.
