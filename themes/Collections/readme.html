Create and Use Collections

Last updated 10/5/2024

Overview

The collection feature allows you to group multiple files together for easy access and management. This is useful when working with related documents or resources that need to be organized in one place.

Step-by-Step Guide

1. Creating a New Collection

To create a new collection:

    Click the Create Collection button to start.

    Select files to include in the collection by clicking Files and choosing from your uploaded files. (Refer to the file feature tutorial above if you need to upload new files.)

2. Naming Your Collection

After selecting the files:

    In the Name field, enter a clear name for your collection (e.g., "Project Documents").

3. Adding a Description

Optionally, you can add a description:

    In the Description field, provide a brief explanation of the collectionâ€™s purpose (e.g., "Collection of all files related to project XYZ").

4. Saving the Collection

Once you have added the necessary files, name, and description:

    Click Save to store the collection. You can now access and manage these files as a group.

Using Collections

Your collections can be used to manage multiple files for projects, tasks, or sharing. You can easily add or remove files from the collection as needed.
