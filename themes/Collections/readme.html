Create and Use Collections

The collection feature allows you to group multiple files together for easy access and management.
This is useful for organizing related documents or resources in one place.

Overview

Building an AI assistant on this platform is intuitive. You can access available assistants or create a new one via the assistant section.

Creating a New Collection

To create a new collection, click the Create Collection button to start. Select files to include in the collection by clicking Files and choosing from your uploaded files.

Naming Your Collection

After selecting the files, in the Name field, enter a clear name for your collection (e.g., "Project Documents").

Adding a Description

Optionally, you can add a description in the Description field to provide a brief explanation of the collectionâ€™s purpose.

Saving the Collection

Once you have added the necessary files, name, and description, click Save to store the collection.

Using Collections

Your collections can be used to manage multiple files for projects, tasks, or sharing. You can easily add or remove files from the collection as needed.
