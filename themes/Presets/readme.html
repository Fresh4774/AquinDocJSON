How to Create and Use a Preset

Last updated 10/5/2024

Overview

The preset feature allows you to define configurations that streamline how the AI interacts with you. Presets can save time by pre-configuring settings such as the model, prompt, and advanced settings like temperature and context length.

Creating a New Preset

To create a preset, click the "Create Preset" button and fill out the necessary fields. You can give the preset a name, select the AI model, and write a prompt that guides the AI's behavior.

Advanced Settings

Adjust advanced settings to suit your needs. You can control the temperature, which defines the creativity of the AI, and the context length, which determines how much of the conversation history the AI retains.

Summary

Presets allow for the creation of custom AI configurations, ensuring that tasks and interactions are carried out efficiently and consistently.
