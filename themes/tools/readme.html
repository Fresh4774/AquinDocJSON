Custom No-Code Tools

Last updated 10/6/2024

Overview

This platform allows you to create custom tools without any coding. You can use these tools to integrate APIs and interact with your own language model (LLM).

Creating a New Tool

To create a new tool, follow these steps:

  • Name: Give your tool a name.
  • Description: Provide a description of what the tool does.
  • Custom Headers: Add custom headers in JSON format.
  • Custom Schema: Define the schema in JSON format.

    • Using Available Tools

    If you have tools available, you can use them within the chat by typing "!" followed by the tool's name or command. This will invoke the tool and allow you to interact with its functionality in real-time.
