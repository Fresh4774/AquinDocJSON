Navigation Button

Last updated: October 4, 2024

Overview

The navigation button allows you to quickly access different features such as chats, presets, prompts, files, collections, assistants, or tools. By clicking on the navigation button, you can easily manage your workspace and organize your tasks without leaving the main screen.

How to Use the Draggable Button

The button is draggable, allowing you to move it around the screen to your preferred position. Click to open the button, drag it by holding the top section, and click again to close it. This flexibility allows you to maintain an organized workspace while accessing all necessary functions.

Managing Multiple Functions

Using the navigation button, you can manage multiple functions simultaneously. Whether you're switching between prompts, presets, or files, this feature ensures seamless multitasking.

Summary

The navigation button is a user-friendly tool designed to streamline your workflow by providing easy access to key features, all within a single button.
